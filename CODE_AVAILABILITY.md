# Code Availability

The source code for GRADV is available in this repository:

https://github.com/alefoxer/VKR_Gertsen_Lenshin

The archived manuscript release will be available on Zenodo after publication of the GitHub release.

The release includes the implementation of audio preprocessing, model adapters, targeted waveform optimization, saliency and occlusion analysis, segment ranking, prototype and spectral stability analysis, model comparison, and the reproducible experiment script.
