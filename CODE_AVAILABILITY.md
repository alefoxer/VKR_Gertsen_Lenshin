# Code Availability

The source code for GRADV is available in the GitHub repository:

https://github.com/alefoxer/VKR_Gertsen_Lenshin

The current archived manuscript release DOI is:

https://doi.org/10.5281/zenodo.20768314

The repository includes the implementation of audio preprocessing, model adapters, targeted waveform optimization, saliency and occlusion analysis, segment ranking, prototype and spectral stability analysis, model comparison, and reproducible experiment scripts.

For manuscript reproduction, the repository includes:

- the final VKR experiment script, `experiments/run_final_vkr_experiment.py`;
- the extended manuscript experiment script, `experiments/run_manuscript_extended_experiment.py`;
- segment-ranking ablation outputs;
- weight-sensitivity analysis outputs;
- model-comparison outputs when comparison adapters are available;
- manuscript-ready figure generation.

The DOI above should be treated as the current archived release DOI until a new GitHub/Zenodo release is created and a new release DOI is issued.
