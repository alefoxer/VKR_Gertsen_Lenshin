# Code Availability

The source code for GRADV is available in the GitHub repository:

https://github.com/alefoxer/VKR_Gertsen_Lenshin

The archived manuscript release is available on Zenodo:

https://doi.org/10.5281/zenodo.20768314

The archived release includes the implementation of audio preprocessing, model adapters, targeted waveform optimization, saliency and occlusion analysis, segment ranking, prototype and spectral stability analysis, model comparison, and the reproducible experiment script.
