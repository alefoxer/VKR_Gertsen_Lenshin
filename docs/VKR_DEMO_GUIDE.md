# Сценарий демонстрации

Этот сценарий помогает быстро показать основные возможности GRADV в интерфейсе.

## 1. Запуск

```powershell
python app.py
```

Откройте:

```text
http://127.0.0.1:7860
```

## 2. Подготовка параметров

Нажмите:

```text
Применить пресет ВКР baseline quality
```

После применения пресета должны быть выбраны:

- модель: `gradv_ru_kws_baseline`;
- целевое слово: `да`;
- сценарий: `Сгенерировать из шума`;
- метод: `Gradient ascent`;
- objective: `logit`.

## 3. Одиночный запуск

Нажмите:

```text
Найти максимизирующий сигнал
```

После завершения покажите:

- карточки `Целевое слово`, `Вероятность до`, `Вероятность после`, `Прирост`;
- паспорт результата;
- итоговое аудио;
- графики waveform, spectrogram и probabilities;
- таблицу сегментной детализации;
- CSV, JSON и WAV-файлы запуска.

Короткое пояснение для пользователя: программа формирует входной сигнал, который повышает score выбранного класса в текущей модели. Сигнал нужен для анализа поведения модели и не обязан звучать как естественная речь.

## 4. Study-режим

В блоке `Исследование образа класса` выберите:

- `Сгенерировать из шума`;
- `Сгенерировать из тишины`;
- 2-3 повтора на условие.

Нажмите:

```text
Запустить исследование образа класса
```

Покажите:

- сводку исследования;
- таблицу всех запусков;
- агрегаты по начальным условиям;
- сравнение запусков;
- лучший найденный образ;
- файлы `study_runs.csv`, `condition_summary.csv`, `study_summary.json`.

## 5. Сравнение моделей

В блоке `Сравнение моделей` выберите нужные модели, например:

- `mock_ru_kws_a`;
- `gradv_ru_kws_baseline`.

Нажмите:

```text
Запустить сравнение моделей
```

Покажите:

- таблицу сравнения;
- график mean/best score;
- success rate;
- CSV и JSON-файлы сравнения.

## 6. Экспорт отчета

После study-запуска нажмите:

```text
Сформировать отчет исследования
```

Отчет сохраняется в:

```text
outputs/reports/<report_id>/
```

## 7. Полный demo-пакет

Для автоматического набора материалов:

```powershell
python experiments\create_vkr_demo.py
```

Результат:

```text
outputs/demo_vkr/<demo_id>/
```

Внутри будут одиночный запуск, study, сравнение моделей, отчет и протокол демонстрации.

## 8. Ограничения

`gradv_ru_kws_baseline` - компактная демонстрационная KWS-модель. Сравнивайте результаты между моделями, начальными условиями и seed; не переносите численные выводы на другие модели без отдельной проверки.
