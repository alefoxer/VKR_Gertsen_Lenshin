# Доступ к приложению

Приложение работает только пока запущен процесс `python app.py` на основном компьютере.

## Локально на этом компьютере

```powershell
python app.py
```

Откройте:

```text
http://127.0.0.1:7860
```

Этот адрес работает только на том компьютере, где запущено приложение.

## С другого устройства в той же сети

Запустите:

```powershell
python app.py --lan
```

В терминале появится адрес вида:

```text
http://192.168.x.x:7860
```

Откройте этот адрес на другом устройстве в той же Wi-Fi/локальной сети.

Если адрес не открывается:

- проверьте, что оба устройства подключены к одной сети;
- временно отключите VPN;
- разрешите Python в Windows Firewall;
- убедитесь, что терминал с приложением не закрыт.

## Временная публичная ссылка

```powershell
python app.py --share
```

Gradio может выдать ссылку вида:

```text
https://xxxxx.gradio.live
```

Ссылка временная. Она работает только пока открыт терминал и запущено приложение.

## Перенос на другой компьютер

Для передачи проекта подготовьте чистый ZIP:

```powershell
python scripts\prepare_github_export.py
```

Готовый архив будет в:

```text
outputs/github_export/
```

На другом компьютере распакуйте архив, установите зависимости и запустите:

```powershell
pip install -r requirements.txt
python app.py
```
